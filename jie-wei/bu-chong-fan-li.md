# 大量練習

## 1 介面 hamburger icon

用 button 標籤做一個 hamburger icon：

JavaScript\(使用 jQuery\)：

```javascript
$(function(){
  /* 按鈕狀態的切換 */
  $("button.hamburger_icon").on("click", function(){
    $(this).toggleClass("-on");
  });
});
```
