# Responsive Web Design 課程

## 什麼是 Responsive Web Design？

又稱做 **RWD**，中譯：響應式設計。  
不論任何裝置，舉凡桌上型電腦、平板、手機都有著不同的螢幕大小，為了解決在同一份網頁面對這些設備，有不同的版面情況、使用情境，因此 RWD 誕生，一切都是為了提供使用者有更好的使用體驗\(User Experience\)：易讀性、操控性。
